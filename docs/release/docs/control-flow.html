Control Flow

If Statements

If statements can be used as expressions or statements. They can also be used for pattern matching, all with a similar syntax. 

    v : 1

    if v is not 1:
        print("nope")
    else
        print("yep")
    end

    -- If as a match statement
    if v is
        0 => print("nope")
        1 => print("yep")
        else => print("wat")
    end

    -- If as an expression
    msg : if v is:
        0 => return "nope"
        1 => return "yep"
        else => return "defintely nope"
    end

    print(msg)

Loops

Only 1 keyword for loops "loop".

Using the "in" keyword, you can specify an integer, float or collection to iterate through or define a new integer, float or collection. 

    loop thing in things:
        print(thing)
    end

    loop c in collection_of_choices:
        if c is
            cool => io(cool value)
            else => io(cool value)
        end
    end

    loop -20 to big_number:
        print("hello")
    end

    -- reverse loop
    loop n in big_number to smaller_number:
        print(n)
    end

    loop item in collection:
        print(item index)
    end
